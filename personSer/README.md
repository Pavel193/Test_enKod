# PersonService
API CRUD сервис для одной модели
- Модель: Person
- Поля модели: Id, email, phone, firstName, lastName
- БД: sqlite 
- Используемая архитектура: https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html
- 4 слоя: 
internal
    ├-app
    ├-logic
    ├-http
    ├-sqlite
